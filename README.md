# LLM vs Markets

A quick look at LLM releases vs major market indices.

Data compiled through Grok 3 and o1, with some manual fact checks.

 ## Example

![Example](./llm_vs_market_example.png)


## Development

```bash
pip install -r requirements.txt
```

## Usage
Start the app with:
```bash
python app.py
```
